# 🌦️ Weather App

Um aplicativo de previsão do tempo desenvolvido com HTML, CSS e JavaScript, criado durante o desafio "30 Days, 30 JavaScript Projects For Beginners" do canal *GreatStack*.

O projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento front-end, especialmente o consumo de APIs utilizando JavaScript puro.

## 📌 Sobre o projeto

O Weather App permite buscar informações climáticas em tempo real através do nome de uma cidade, exibindo dados como temperatura, umidade, velocidade do vento e condição climática atual.

Durante o desenvolvimento, foram aplicados conceitos importantes como:

- Requisições HTTP com fetch()
- Programação assíncrona com async/await
- Manipulação do DOM
- Consumo de APIs REST
- Tratamento de erros
- Manipulação de dados JSON


## 🚀 Funcionalidades

✔️ Busca de clima por cidade

✔️ Exibição de temperatura em tempo real

✔️ Informações de umidade e vento

✔️ Ícones dinâmicos conforme a condição climática

✔️ Mensagens de erro para cidades inválidas

✔️ Interface simples e responsiva

## 🛠️ Tecnologias utilizadas
* Tecnologia	Descrição
* HTML5	Estrutura da aplicação
* CSS3	Estilização e responsividade
* JavaScript	Lógica da aplicação
* OpenWeatherMap API	Fonte dos dados climáticos
